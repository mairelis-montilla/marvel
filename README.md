## Setup del Proyecto

Clonar repositorio
```
$ git clone https://github.com/mairelis-montilla/sorting-carrier.git
```

Luego realizar el yarn install
```
$ yarn install
```

Para compilar y levantar el servidor en local
```
$yarn serve
```
La aplicacion correra en el puerto 8080
